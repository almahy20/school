```
✅ واضح: "اذهب لتفاصيل الفصل ← أدر المنهج"
```

```
✅ كل فصل ← منهجه الخاص
❌ مناهج عامة ← ربط يدوي بالفصول
```

### 3. تقليل التعقيد
```
✅ صفحة واحدة للتعديل
❌ صفحتين需要同步
```

### 4. تقليل الكود
```
✅ حذف 548 سطر غير ضروري
❌ كود مكرر في صفحتين
```
