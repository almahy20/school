# إدارة عربية — Edara Arabiya

نظام إدارة المدارس الذكي المتكامل.

## المميزات
- إدارة الطلاب والمعلمين.
- رصد الحضور والغياب والدرجات.
- نظام إشعارات فورية (Push Notifications).
- إدارة المناهج والجداول الدراسية.
- لوحة تحكم للمدير، المعلم، وولي الأمر.

## التقنيات المستخدمة
- React + Vite
- TypeScript
- Tailwind CSS
- Supabase (Backend & Auth)
- shadcn/ui

## التشغيل المحلي
```sh
npm install
npm run dev
```
