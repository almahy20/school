# إدارة عربية — Edara Arabiya

نظام إدارة المدارس الذكي المتكامل.

## المميزات
- إدارة الطلاب والمعلمين.
- رصد الحضور والغياب والدرجات.
- نظام إشعارات فورية (Push Notifications).
- إدارة المناهج والجداول الدراسية.
- لوحة تحكم للمدير، المعلم، وولي الأمر.

## التقنيات المستخدمة
- React + Vite
- TypeScript
- Tailwind CSS
- Supabase (Backend & Auth)
- shadcn/ui

## هيكلية المشروع المحدثة
تمت إعادة هيكلة المشروع لتبسيط الأكواد وتحسين القابلية للصيانة:
- `src/components/class-detail/`: يحتوي على المكونات الفرعية لصفحة تفاصيل الفصل.
- `src/components/database/`: يحتوي على مكونات إدارة قاعدة البيانات.
- `src/config/`: يحتوي على ملفات التكوين الثابتة.
- `docs/`: يحتوي على التوثيق التقني المفصل للإصلاحات والميزات.

```sh
npm install
npm run dev
```
